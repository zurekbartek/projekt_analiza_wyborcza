# Analiza poparcia politycznego i przyrostu naturalnego w Polsce (2023)

Projekt służy do analizy zależności pomiędzy:
- wynikami wyborów parlamentarnych 2023 (poparcie partii w powiatach),
- przyrostem naturalnym w powiatach,
- strukturą zwycięstw partii (1., 2. i 3. miejsce).

Wyniki prezentowane są w formie map tematycznych Polski.
